# Python CI/CD using GitHub Actions 🚀

This project demonstrates how to use GitHub Actions for automating a Python project's workflow using a CI/CD pipeline.

---

## 🚀 Features
- ✅ Linting with `flake8`
- ✅ Testing with `pytest`
- 🔁 Auto-run on push and pull requests to `main` branch
